# Python tooling
- use `uv` for various Python related actions
- use `uv run python <python-file>` to run application
- use `uv run pytest -v` to run test

# MCP Tools
- use `context7` to access external documents for libraries/frameworks (e.g., `Next.js`, `google-adk-python`, `fastapi`)
- use `serena` to explore code base efficiently
- use `deepwiki` to conduct deep dive into external libraries/frameworks (e.g., `Next.js`, `google-adk-python`, `fastapi`)

# manifest tool
- The manifest_append tool expects a JSON string, not a YAML string.
